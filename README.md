
#  Projeto de Cadastro de Usuários — Acompanhando o Rodolfo Mori

Este projeto foi desenvolvido acompanhando as aulas do **Rodolfo Mori**, com o objetivo de aprender na prática como criar uma aplicação **Full Stack** — com **front-end em React** e **back-end com Node.js, Express, Prisma e MongoDB**.  
Durante o desenvolvimento, aprendi a **conectar o front com o back**, criar uma **API funcional** e trabalhar com **requisições HTTP** utilizando o **Axios**.

----
## Tecnologias utilizadas

###  Back-end
- **Node.js**  
- **Express**  
- **Prisma ORM**  
- **MongoDB (Atlas)**  

###  Front-end
- **React (Vite)**  
- **Axios**  
- **CSS3**

---

##  Funcionalidades

- Criação de usuários (com nome, idade e e-mail)  
- Listagem automática dos usuários cadastrados  
- Exclusão de usuários com ícone de lixeira  
- Integração total com o banco de dados MongoDB  

---

##  Aprendizados

Esse projeto foi uma ótima oportunidade para entender melhor **como o front-end se comunica com o back-end** e como os dados fluem entre cliente, servidor e banco de dados.  

Também aprendi sobre **estruturas de pastas**, **rotas HTTP** e **requisições assíncronas com Axios**.  

Um dos pontos mais desafiadores foi o **erro de CORS**, que me fez pesquisar bastante até entender o que realmente estava acontecendo.

---



##  O que é o erro de CORS?

O erro **CORS (Cross-Origin Resource Sharing)** acontece quando o **navegador bloqueia uma requisição HTTP** por segurança — isso ocorre quando o **front-end e o back-end estão em endereços diferentes**.
